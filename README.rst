font-subset-css
=================================

Requirements
------------

-  Python >= 3.6

Installation
------------

.. code:: sh

    $ pip install font-subset-css

Usage
------------

.. code:: sh

    $ fontsubsetcss YOUR_FONT.otf
    $ fontsubsetcss YOUR_FONT.otf  --output-dir style
    $ fontsubsetcss YOUR_FONT.otf  --text-file YOUR_TEXT.txt
    $ fontsubsetcss YOUR_FONT.otf  --text 'ABCD'
